🌸 Air Canvas – A Small Women's Day Project

This is a small computer vision experiment built using Python and OpenCV where you can draw on the screen by moving your hand in front of the webcam.

The idea was to create a simple virtual canvas that tracks movement and lets the user draw in the air.

To celebrate Women's Day, the program displays a message wishing Happy Women's Day to all the amazing women in tech.

---

✨ Features

- Webcam-based drawing using OpenCV
- Real-time canvas display
- Simple gesture/movement based drawing
- On-screen Happy Women's Day message

---

🛠 Technologies Used

- Python
- OpenCV
- NumPy

---

🚀 How to Run

Install dependencies:

pip install opencv-python numpy

Run the program:

python air_canvas.py

---

💭 About This Project

This started as an attempt to build an air drawing canvas where I could write “Happy Women's Day” by moving my hand in the air.

Reality check: handwriting with computer vision is much harder than it sounds.

Instead of beautifully writing text, the program mostly produced random shapes and chaotic lines. After several attempts, I decided to keep the drawing feature simple and just display the message on screen.

Even though the result is small, the project was a fun way to experiment with computer vision and real-time image processing.
